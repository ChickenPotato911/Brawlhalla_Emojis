# Brawlhalla_Emojis
Download All Brawlhalla Emojis In An Organized Collection, 
Sourced From The Brawlhalla Wiki, 
See Here For More: https://brawlhalla.wiki.gg/wiki/Emojis#Emoji_Skins, 
A Few Animated Emojis Are Currently Static Images But May Be Updated To GIFs In The Future, 
Some Newer 2026 Emojis Are Excluded For Now But Could Be Added Later, 
Contributions Or Suggestions For Missing Emojis Are Most Welcome! 👍



## Installation

### **Option 1: Download Manually**
1. Click The Green **"Code"** Button Above.
2. Select **"Download ZIP"** To Get All Emojis In A Single Folder.
3. Extract The ZIP File And Use The Emojis As Needed.

### **Option 2: Clone The Repository (Git Required)**
```bash
git clone https://github.com/ChickenPotato911/Brawlhalla_Emojis.git
```

Do Note That The Emojis Are Oganized Into Subfolders
